# AI Blog Generator

AI-powered web application that generates blog content based on user-provided topics using Large Language Models (LLMs).

The application allows users to quickly create coherent, structured blog posts with a clean and responsive user interface. It demonstrates full-stack development skills, API integration, and deployment.
